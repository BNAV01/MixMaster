# Documents

Esta carpeta concentra la documentación viva del proyecto.

- `product/`: visión, módulos, roadmap.
- `architecture/`: decisiones y estrategias de arquitectura.
- `database/`: modelo de datos y lineamientos relacionales.
- `ux/`: principios y experiencia de producto.
- `api/`: convenciones y contratos de integración.
- `adr/`: decisiones de arquitectura registradas.
