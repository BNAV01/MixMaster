# Estrategia de Feedback

## Objetivo

Cerrar el loop entre recomendación, consumo y mejora del sistema.

## Principios

- corto
- contextual
- no invasivo
- accionable

## Modelo UX

- pregunta inicial: `¿Te acertamos?`
- respuestas: `Sí`, `Más o menos`, `No`
- ajuste opcional: `Más dulce`, `Menos dulce`, `Más suave`, `Más intenso`, `Más parecido a lo mío`, `Más atrevido`
